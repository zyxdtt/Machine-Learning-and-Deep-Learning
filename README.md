# Machine Learning vs Deep Learning on MNIST

A small research project comparing the performance of Machine Learning and Deep Learning approaches on the MNIST digit classification task.

## 📌 About This Project

This repository contains a Python script (about 200 lines) that explores an interesting question: **"When it comes to MNIST digits, can Deep Learning really outperform traditional Machine Learning?"**

The code includes all necessary dependencies, which can be easily installed. Simply run the script to discover the answer yourself!

## 🎓 Background

As a freshman at a general university, this is my humble attempt to explore the differences between these two important approaches in artificial intelligence. While this research may not be perfectly rigorous or standardized, I've tried my best to conduct a fair comparison.
